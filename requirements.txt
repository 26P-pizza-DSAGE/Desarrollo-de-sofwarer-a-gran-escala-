# Proyecto Python 3.13 en Docker

Este repositorio incluye una configuración de **Dev Containers** para que puedas programar sin necesidad de instalar Python en tu máquina local.

## 🚀 Cómo usarlo

1. Clona este repositorio o descarga el código.
2. Abre la carpeta en **VS Code**.
3. Instala la extensión **Dev Containers** de VS Code.
4. Presiona `F1`, escribe `Dev Containers: Reopen in Container` y selecciónalo.
5. ¡Listo! Abre una terminal en VS Code y ejecuta `python app.py`.
